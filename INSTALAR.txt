INSTALACIÓN EN CLAUDE CODE
===========================

Opción 1: DIRECTA (solo este proyecto)
------
1. Abre esta carpeta en Claude Code:

   cd kit-visionloop-replicator
   claude

   O: File → Open Folder → selecciona esta carpeta

2. Escribe cualquier cosa en el chat:

   Hola

3. Claude te dará la bienvenida y activará la skill automáticamente.

4. Envía tu input:

   Replica https://linear.app

   O arrastra un screenshot (PNG/JPG), o pega HTML.

5. Espera 5-25 minutos. Claude genera el proyecto solo.


Opción 2: GLOBAL (instala para todos tus proyectos)
------
1. Copia el archivo de skill a tu carpeta global:

   Cómo copiar:

   - Navega a: kit-visionloop-replicator/.claude/skills/

2. Reinicia Claude Code

3. Ahora en CUALQUIER proyecto, di:

   Replica https://ejemplo.com


INSTALACIÓN EN VSCODE
======================

Una vez que la skill generó la carpeta "replica/":

1. Abre el proyecto:

   cd replica
   code .

2. En VSCode se abre automáticamente.

3. Levanta el dev server en la terminal integrada:

   pnpm dev

   (si no tienes pnpm: npm install -g pnpm)

4. Abre http://localhost:3000 en el navegador

5. Los cambios en el código se ven al guardar (hot reload)


CÓMO USAR
==========

BÁSICO (todo automático):
------------------------
claude               # Abre Claude Code en la carpeta
Replica https://...  # Dile qué replicar
[espera 5-25 min]    # Claude itera solo hasta ≥99% similitud
cd replica && pnpm dev  # Ver el resultado en el navegador


PERSONALIZAR (editar código):
-----------------------------
cd replica
code .                                    # Abre VSCode

# Edita components/sections/*.tsx
# Edita lib/design-tokens.ts para colores
# Cambios se ven al guardar (hot reload)


DEPLOYAR (subir a internet):
----------------------------
cd replica
npx vercel                               # Sube a Vercel en 1 minuto


REQUISITOS MÍNIMOS
===================

✓ ~600 MB libres en disco
  Para: Next.js + assets + node_modules

✓ Claude Code instalado
  Descargar: https://claude.ai/code

✓ Internet (para crawling de URLs)
  Sin requerimiento si usas screenshot/HTML

HELP RÁPIDO
===========

P: ¿Necesito instalar Playwright manualmente?
R: No. La skill lo hace automáticamente.

P: ¿Cómo cambio colores/tipografía?
R: Edita lib/design-tokens.ts

P: ¿Cómo agrego nuevas secciones?
R: Crea components/sections/MiSección.tsx e importa en app/page.tsx

P: El loop se queda atascado
R: Normal. Máx 25 iteraciones (~20-30 min). Tras 3 sin mejora, rehace el componente.

P: Lighthouse no llega a 90
R: Pide "optimiza lighthouse del replica" y Claude aplica fixes.

P: Quiero editar la réplica generada
R: Es código editable standard Next.js. Toca lo que necesites.

P: Quiero deployar
R: cd replica && npx vercel (1 minuto en Vercel)

SOPORTE
=======

Si algo falla:

1. Verifica Node.js:
   node -v          (debe ser ≥20)

2. Limpia y reinstala:
   rm -rf node_modules .next
   pnpm install

3. Intenta de nuevo:
   pnpm dev

4. Si sigue fallando:
   Pídele a Claude que debuggee el error exacto
